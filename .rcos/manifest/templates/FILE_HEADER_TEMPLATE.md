# File Header Template

Add a concise structured header to important code files to make authority and purpose explicit.

```python
"""
Module: [module name]

Purpose:
- [purpose 1]
- [purpose 2]

Authority:
- [logic area this file owns]

Depends on:
- [dependency]
- [dependency]

Do not:
- [forbidden pattern]
- [forbidden pattern]

Used by:
- [file/module]
- [file/module]
"""
```

Notes:

- Keep it short and factual.
- Do not use aspirational language.
- Update it when authority changes materially.
